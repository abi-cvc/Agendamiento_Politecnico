🚀 Prueba API REST - Cancelar Citas

Interfaz de prueba para los endpoints REST

ℹ️ Información: Esta página permite probar los endpoints REST de cancelación de citas. Ingresa un ID de cita válido (por ejemplo: 1, 2, 3, etc.)

📋 1. Obtener Confirmación de Cancelación

GET /rest/api/citas/{idCita}/cancelar/confirmacion

❌ 2. Confirmar Cancelación

POST /rest/api/citas/{idCita}/cancelar

🔄 3. Consultar Estado de la Cita

GET /rest/api/citas/{idCita}
